# Nukepedia
Nukepedia는 뉴크 개발자들이 자신이 만든 기능을 공유하는 사이트 입니다.
대부분 많은 회사들이 이곳에서 정보를 얻어 필요한 기능을 회사에 설치하여 사용합니다.

## 가입

http://www.nukepedia.com/

위 사이트에 들어가서 회원가입을 해주세요.
