# Gaffer

라이팅툴이지만 데이터 확인용으로도 편리합니다.

홈페이지 : http://www.gafferhq.org
## 설치

```
$ cd ~/app
$ wget https://github.com/GafferHQ/gaffer/releases/download/0.50.0.0/gaffer-0.50.0.0-linux.tar.gz
$ tar -xzf gaffer-0.50.0.0-linux.tar.gz -C ~/app
```
