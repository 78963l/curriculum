# Crontab
컴퓨터에 일정시간마다 실행되어야 하는 프로세스를 등록할 때 사용합니다.

### 예약하기
터미널에 아래처럼 타이핑합니다.

```
crontab -e
```
vi 에디터가 열립니다.

### 휴식시간 알림.
월~금 10시부터 18시까지 매 50분이 되면 breaktime을 출력하는 명령어입니다.
wall 명령어는 터미널로 브로드 캐스트 메시지를 보내는 명령입니다.
```
50 10-18 * * 1-5 echo "breaktime" | wall
```
