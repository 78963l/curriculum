# ffmpeg

```
# yum install ffmpeg
```

### 이미 빌드되어있는 ffmpeg 설치하기
```
$ su
# cd /usr/local
# mkdir ffmpeg
# cd ffmpeg
# wget http://johnvansickle.com/ffmpeg/builds/ffmpeg-git-64bit-static.tar.xz
# tar xpvf ffmpeg-git-64bit-static.tar.xz --strip 1
```
