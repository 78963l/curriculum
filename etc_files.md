# 기타 설정에 자주 사용되는 파일
많은 툴에서 설정파일로 사용되는 기타 파일들을 알아보겠습니다.

자신이 만든 프로그램의 설정파일을 저장해야하는 상황이 발생할 수 있습니다.
이때 자신만의 포멧을 만들기 보다는 다른 프로그래머들이 쉽게 해당 설정 데이터를 가져다 사용할 수 있도록
범용 포멧으로 저장하는 것을 추천합니다.
또한 많은 툴들이 이미 범용 포멧을 사용하여 저장하고 있습니다.
